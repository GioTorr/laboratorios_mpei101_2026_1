# Practicas de Laboratorios MPEI-101 2026-1
**Universidad Escuela Colombiana de Ingeniería Julio Garavito**
Repositorio de prácticas de laboratorio desarrolladas durante el primer semestre de 2026 para la asignatura Microprocesadores e Interfaces (MPEI), grupo 101, bajo la tutoría del docente a cargo.

## Tecnologías Utilizadas
* Git / GitHub: Control de versiones.
* STM32CubeIDE: Entorno de desarrollo para microcontroladores STM32.
* STM32CubeMX: Configuración de periféricos y generación de código.
* Proteus: Simulación de circuitos electrónicos y hardware.
